# city-border

Detect the border of urban areas. Please see [1](http://iopscience.iop.org/article/10.1088/1742-5468/2016/05/053205/meta) or [2](https://arxiv.org/abs/1510.06326) for a description of the method.

### Dependencies
* numpy (1.15.1)
* scipy (1.0.0)
* matplotlib (2.2.2)
* osmnx (0.8.1)
* networkx (2.1)
* opencv (3.4.1)
* scikit-image (0.13.0)
